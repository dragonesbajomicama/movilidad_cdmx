Movilidad CDMX

Este proyecto explora la movilidad urbana desde la escala del caminante. Caminar se entiende no solo como un desplazamiento, sino como una forma de lectura del territorio, donde el cuerpo, el espacio y las condiciones del entorno producen conocimiento.

La investigación se documenta mediante caminatas registradas en bitácoras, fotografías y observaciones situadas. Estos registros se traducen posteriormente a soportes materiales, particularmente tejido y bordado, como una forma de sistematización visual de trayectos, patrones y anomalías del recorrido urbano.
